# Background Eraser - 배경 제거 도구

이미지의 배경을 클릭 한 번으로 쉽고 빠르게 투명하게 만드는 웹 애플리케이션입니다. 모든 작업은 브라우저 내에서 이루어지므로 이미지가 서버로 전송되지 않아 보안이 우수합니다.

## ✨ 주요 기능

- **클릭 기반 배경 제거**: 투명하게 만들고 싶은 영역을 클릭하면 유사한 색상의 배경이 자동으로 제거됩니다. (Flood Fill 알고리즘 적용)
- **허용 오차(Tolerance) 조절**: 색상 허용 오차를 조절하여 더 넓거나 좁은 범위의 배경을 한 번에 제거할 수 있습니다.
- **실행 취소(Undo) 및 초기화**: 실수로 잘못 제거한 경우 이전 단계로 되돌리거나 처음 상태로 초기화할 수 있습니다.
- **확대/축소 지원**: 정밀한 작업을 위해 이미지를 확대하거나 축소하여 확인할 수 있습니다.
- **PNG 내보내기**: 배경이 투명해진 이미지를 PNG 형식으로 다운로드할 수 있습니다.

## 🚀 사용 방법

1. `index.html` 파일을 웹 브라우저로 엽니다.
2. 배경을 제거할 이미지를 드래그하거나 클릭하여 업로드합니다.
3. 투명하게 만들고 싶은 배경 영역을 클릭합니다.
4. 필요에 따라 '색상 허용 오차' 슬라이더를 조절하여 제거 범위를 변경합니다.
5. 작업이 완료되면 'PNG로 다운로드' 버튼을 클릭하여 결과물을 저장합니다.
